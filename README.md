# Interest Explorer ==>

Interest Explorer is a simple web application that helps users understand how compound interest works. By changing the investment amount, interest rate, and number of years, users can see how their money grows over time.

## Features

- Change the starting amount using a slider
- Adjust the annual interest rate
- Select the investment period
- View the final amount instantly
- See total interest earned
- Interactive chart for better visualization
- Simple and easy-to-use interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

## What I Learned

While building this project, I learned:

- How to work with JavaScript event listeners
- DOM manipulation
- Using formulas in JavaScript
- Creating charts with Chart.js
- Making interactive web applications

## How It Works

The calculator uses the compound interest formula to calculate the future value of an investment.

Users can adjust the values and the results update automatically without refreshing the page.
